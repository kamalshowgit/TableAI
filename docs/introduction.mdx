## What is TableAI?

TableAI is a secure, local AI-powered data analysis tool that helps you explore and understand your data through natural language conversations. Upload your files or connect to databases, then ask questions in plain English to get instant insights, visualizations, and analysis.

## Key Features

- **🔒 Privacy First**: All processing happens locally using Ollama
- **📊 Smart Analysis**: Powered by Mistral LLM and LlamaIndex
- **🗄️ Multiple Sources**: Files (CSV, Excel, JSON, Parquet) and databases
- **🎨 Beautiful UI**: Modern, responsive interface with dark mode
- **⚡ Fast Performance**: Cached models for quick responses
- **🛡️ Safe Execution**: Sandboxed code execution with security filters

## Supported Data Sources

### File Formats
- CSV (Comma-separated values)
- Excel (XLSX, XLS)
- TSV (Tab-separated values)
- Parquet
- JSON

### Databases
- SQLite
- PostgreSQL
- MySQL
- MariaDB
- SQL Server
- Oracle
- DuckDB
